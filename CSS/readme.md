# CSS - Cascading Stylesheet

# Type
* Internal
  - Embedded (Using style tag)
  > <pre>&lt;style&gt;selector {property: value}&lt;/style&gt;</pre>
  - Inline (Using style attribute)
  > <pre>&lt;p style="property: value; .."&gt;&lt;/p&gt;</pre>

* External - (Using link tag)
> <pre>&lt;link rel="stylesheet" href="path/to/file.css" /&gt;</pre>

# CSS Selectors
* Tag Selector
  > <pre>tag_name {property: value}</pre>
* ID Selector
  > <pre>#id_value {property: value}</pre>
* Class Selector
  > <pre>.class_value {property: value}</pre>
* Attribute Selector
  > <pre>tag_name[attr=xyz] {property: value}</pre>
* Psuedo Selector
  > <pre>::psuedo_property {property: value}</pre>
